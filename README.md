# 🏀 Site Web Officiel du Combourg Basket

Bienvenue sur le dépôt du site web officiel du club de basket-ball de Combourg ! Ce projet a pour objectif de fournir une plateforme moderne et dynamique pour l'équipe, permettant aux utilisateurs de suivre l'actualité, de consulter les équipes, les résultats, et d'accéder à la boutique.

---

## 🚀 Démarrer le Projet

Ce projet est une application [Next.js](https://nextjs.org) (basée sur React) et utilise [Tailwind CSS](https://tailwindcss.com) pour le stylisme.

### 📦 Installation et Lancement

Pour cloner le dépôt et lancer l'application en local, suivez ces étapes :

1.  **Cloner le projet :**
    ```bash
    git clone [https://github.com/Antonito35/combourgchateaubriandbasketv2.git](https://github.com/Antonito35/combourgchateaubriandbasketv2.git)
    cd combourgchateaubriandbasketv2 
    ```

2.  **Installer les dépendances :**
    ```bash
    npm install
    ```

3.  **Démarrer le serveur de développement :**
    ```bash
    npm run dev
    ```

Le site sera accessible dans votre navigateur à l'adresse suivante : **[http://localhost:3000](http://localhost:3000)**

---

## ✨ Fonctionnalités Clés

* **Actualités et Événements** : Affichage des événements, matchs et dernières nouvelles du club.
* **Équipes et Joueurs** : Gestion et présentation des différentes équipes et de leurs membres.
* **Résultats et Classements** : Suivi des performances, scores et positionnement des équipes.
* **Boutique en Ligne** : Plateforme d'achat d'articles personnalisés (avec options de flocage).
* **Média** : Galerie photos et vidéos (images du terrain, des matchs, etc.).

---

## 🛠️ Technologies Utilisées

* **Framework** : [Next.js](https://nextjs.org) (React)
* **Langage** : JavaScript, HTML, CSS
* **Stylisme** : [Tailwind CSS](https://tailwindcss.com) (pour un développement rapide et modulaire)
